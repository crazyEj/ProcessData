# ProcessData

## Run the app
From the workspace root folder, start the app with:

`py -m streamlit run app.py`

This launcher script automatically loads the real Streamlit application from `Data_Triage_App/app.py`.

ProcessData is an adaptive data triage & predictive studio. It automatically ingests raw datasets, cleans structural errors, handles fuzzy text/outliers via ML, and runs auto-configured ML models (Regression/Classification). It features advanced ARIMA time-series forecasting with 95% confidence bands to map pricing trend velocities.
